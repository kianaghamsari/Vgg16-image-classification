# Image Classification using VGG16

This project implements an image classification pipeline using transfer learning
with the VGG16 convolutional neural network.

> This project was originally developed as part of a university course.

## Highlights
- Transfer learning with pre-trained VGG16 backbone
- Fine-tuned classifier for custom image dataset
- Performance improvement using dropout regularization

## Tech Stack
- Python
- PyTorch
- torchvision
